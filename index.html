Cómo recorrer y editar

Galería virtual 3D con editor de planta propio.

Moverse

  • Clic en la vistaBloquea el cursor para mirar con el ratón.
  • W A S D · ShiftCaminar / correr.
  • EscLibera el cursor para usar el panel.
  • TabAlternar Recorrido y Edición.

Recorrido

  • ApuntarMuestra título, fecha y descripción.
  • ClicAbre a pantalla completa; la descripción tiene scroll.

Edición

  • Subir / Texto + clicColoca imágenes o textos en las paredes.
  • RuedaEscala la selección.
  • Flechas / G / MMover · agarrar y deslizar · reubicar. Q/E giran los elementos.
  • Ctrl+clicSeleccionar varias obras a la vez.
  • SuprEliminar la selección.
  • PanelColor por sala, pared individual, altura, iluminación, marco, datos y texto.

Atajos de teclado

  • Ctrl+Z / Ctrl+YDeshacer / rehacer (también Ctrl+Shift+Z).
  • Ctrl+C / Ctrl+VCopiar y pegar la selección.
  • Ctrl+DDuplicar la selección en el sitio.
  • Ctrl+ASeleccionar todas las obras.
  • Ctrl+SGuardar en el navegador.

Arrastra para crear la forma elegida · clic coloca una luz · con Ruta haz clic para añadir puntos del recorrido guiado (saldrán como flechas en el piso) · con Seleccionar: arrastra los vértices (cuadrados) para deformar, los puntos medios (círculos) para curvar la pared, doble-clic en una arista añade un vértice y en un vértice lo elimina · en las propiedades de cada sala puedes quitar el techo y cargar texturas de paredes, techo y piso · rueda zoom · arrastra el fondo para desplazar · Borrar elimina salas, luces o puntos de ruta.
